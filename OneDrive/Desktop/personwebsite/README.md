# Personal Website - HAj 

A modern, responsive HAJ website template for a web developer portfolio.

## Features

- Clean and modern design with a vibrant red and dark blue color scheme
- Fully responsive layout that works on all devices
- Smooth scrolling navigation
- Social media integration
- Professional illustration
- Contact information display

## Getting Started

1. Open `index.html` in your web browser
2. Customize the content, links, and information to match your personal details

## Customization

### Update Personal Information

1. **Name**: Edit the name in `index.html` 
2. **Email**: Update the email address in both the social bar and email bar sections
3. **Social Links**: Update the social media links in the social bar section
4. **Profession**: Change "Web Developer" to your profession

### Update Colors

Edit the color scheme in `styles.css`:
- Red accent: `#E74C3C`
- Dark blue background: `#1A1A2E`
- White text: `#FFFFFF`

### Add Sections

To add About, Portfolio, or Blog sections, create new sections in `index.html` and link them in the navigation.
